# 财务 BI 分析仪表盘 (Financial BI Dashboard)

这是一个面向企业级财务分析的轻量化、高性能 BI 仪表盘系统，支持从数据清洗、多维过滤、交互式可视化到协同分析的完整闭环体验。

---

## 核心功能 (Core Features)

### 📊 1. 动态图表与下钻分析 (Interactive Dashboards)
- **多图表无缝切换**：提供柱状图（多维指标对比）、折线趋势图（单月历史趋势）以及饼图（单期数据分布占比）。
- **智能指标管控**：饼图模式下自动过滤非占比指标，趋势图强制应用单月发生额逻辑，确保分析口径严谨。
- **逐级下钻 (Drill-down)**：图表与表格深度联动，点击数据节点可一键在多维表中定位详情。

### 📈 2. 深度多维交叉表 (Advanced Matrix)
- **双级 Y 轴钻取**：支持主、次两级维度嵌套（如：业态 + 管理口径），并自动实现 Row-spanning 跨行合并展示。
- **X 轴行列互换**：提供 `X-Axis Swapped` 功能，可一键切换“指标优先”或“时间组优先”视图。
- **自定义业务排序**：针对核心维度内置行业级排序逻辑（业态、管理口径、产权口径等）。
- **指标全量管控**：支持跨计算组指标多选，提供“全选/清空”快捷操作，适应大宽表场景。

### 📄 3. 专业级报表导出 (Enterprise Reporting)
- **横向分页打印 (Horizontal Pagination)**：针对过宽的财务报表，系统会自动进行横向切分（Chunking），确保 PDF 每一页都清晰美观。
- **页脚自动重复 (Sticky Footer)**：打印时，表格底部的“合计行”会自动固定在每一页底部，方便跨页对比。
- **红绿配色方案**：同比/环比差额指标自动应用“红降绿增”配色，符合财务汇报习惯。
- **精度智能控制**：支持“整数模式”与“默认模式”切换，增长率指标自动提供 1 位小数精度及趋势纠偏。

### 📝 4. 业务评论看板 (Comments Board)
- **多维联动筛查**：支持通过全量维度过滤历史批注记录。
- **智能上下文补全**：在线填报时自动匹配项目对应的业态与口径，实现零差错数据录入。
- **Excel 模板导入**：基于 `exceljs` 生成自带下拉校验的官方模板，支持大规模批量批注上报。

## 📖 使用说明 (Usage Guide)

### 如何使用多维交叉表？
1.  **设置维度**：在左侧切片器选择“数据表维度”，并可选“二级 Y 轴”进行深度拆解。
2.  **切换视图**：点击右侧的 **"X-AXIS SWAP"** 按钮，可以快速在“指标分组”和“计算项分组”之间切换。
3.  **调整精度**：点击 **"DEFAULT VIEW / INTEGER MODE"** 按钮，可以在显示 1 位小数（推荐查看增长率）和全整数视图之间切换。
4.  **保存方案**：调整好满意的筛选组合后，点击“保存方案”，下次登录可一键恢复当前视角的全部配置。

### 打印 PDF 注意事项
*   点击 **"PRINT PDF"** 前，请确保已勾选所有需要的指标。
*   打印预览中，系统会自动处理横向分页。如果指标很多，预览可能显示多部分，这是为了保证打印质量。
*   合计行会出现在每一页的末尾，无需手动翻页查找。

## 🛠 技术栈 (Tech Stack)

- **前端**：React 19 + TypeScript + Vite + Tailwind CSS
- **可视化**：Recharts / Chart.js
- **文档处理**：SheetJS (XLSX) / ExcelJS / FileSaver.js
- **后端支持**：Supabase (PostgreSQL) 提供云端数据持久化、权限管控及预设方案存储。

## 📋 数据规范 (Data Specification)

*   **数据源**：主要依赖于项目根目录或上传的 Excel 报表。
*   **计算口径**：内置利润率、用工成本率、能源成本率等核心指标计算逻辑。
*   **异常值处理**：对于分母为 0 或计算错误的指标，系统会自动显示为 `-`；对于超过 10000% 的极端值，显示为 `10000%+` 以保持界面整洁。
