月末结账及报表注意事项

请按步骤依次完成结账工作，确保各科目数据勾稽无误。

一、薪酬业务

01 MODULE
  • 完成本月薪酬业务计提、发放及相关凭证生成，包括工资、奖金、社保、公积金、福利费、工会经费及职工教育经费等业务处理。
  • 检查涉及薪酬业务的凭证中增减变动原因是否正确。
  • 检查用工类型是否选择正确，不同用工类型余额是否合理，是否存在串项情况。
  • 检查应付职工薪酬明细科目中是否存在不合理借方余额。
  • 薪酬报表检查：在报表系统中取数并校验薪酬报表及相关损益表数据是否准确。

二、租赁业务

02 MODULE
  • 完成租赁合同评估记账、当月折旧计提、资产清算对账、付款清算对账工作。
  • 租赁报表检查：在报表系统中取数并校验租赁相关报表及相关资产负债表数据是否准确。

三、资产业务

03 MODULE
  • 完成资产业务单据检查、折旧检查、月末检查、资产提满折旧日期处理、数据一致性检查等工作。
  • 资产报表检查：在报表系统中取数并校验资产相关报表及相关资产负债表数据是否准确。

四、往来资金业务

04 MODULE
  • 内部款项对账：与上级资金单位对账，检查内部存款、内部存入款余额中贸易伙伴是否正确。
  • 检查客户、供应商挂账及往来单位余额是否准确，防范出现不合理负数余额，确保账龄与逾期反映真实业务。
  • 银行存款检查：核对现金流量项目、贸易伙伴及期限管理是否准确；确保各银行账户无不合理贷方余额。
  • 内部交易实时监控：每日通过系统事务代码 ZP0JYADG0030 检查内部往来及内部暂估交易，提前消除红灯差异。

五、总账业务

05 MODULE
  • 外币评估：外币业务单位必须在月末通过ERP系统标准化流程完成外币评估（汇率参考月末最后一天）。
  • 坏账计提：数据无变化后计提本月坏账金额，重点核查并修改往来业务性质和是否逾期。
  • 减资准备表校验：校验 BA07 报表本年计提数是否与利润表“信用减值损失+资产减值损失”相等。

六、涉税业务

06 MODULE
  • 完成所有税费计提与缴纳凭证处理，检查凭证中增减变动原因，纠正不合理借方余额。
  • 税费报表检查：取数校验税费报表，检查 BC02 (税金及附加计提) 与 BB41 的勾稽关系是否通过。

七、维修业务

07 MODULE
  • 维修工单停止：处理并结算本月维修工单，停止系统里新的工单提报，未办完单据顺延下月。
  • 财务部门根据业务办理结果完成本月维修工单的结算及结转凭证生成。

八、项目月结

08 MODULE
  • 投资项目结转：检查本月投资业务入账情况，确认完成后将投资项目结转至在建工程科目。
  • 成本项目结转：维修结算后检查项目结转规则，核对项目发生额，结转至功能范围（注：功能范围6601和6602不用结转）。

1. 资产负债表

BA01
  • 防止合同资产重分类错误：严禁因合同负债清账错误导致往来单位出现借方余额，从而在期末被系统自动重分类至合同资产中。
  • 科目列报明细管控：“一年内到期的非流动负债”科目中，除“一年内到期的租赁负债”明细外，其余项严禁包含任何余额数。

2. 现金流量表

BA04
  • 现金流项目实质选择：按业务实质参考标准选择现金流项目，严禁将代付款项错选为薪酬或分红支出。
  • 党费户受限资金管理：独立核算的党费账户收付款项，在期限管理中必须选择“C98-受限存款”，防止影响期末现金等价物期末余额。
  • 关联往来单位字段录入：中石油及华油系统内交易，需在“银行存款”科目的往来单位和贸易伙伴字段中维护对应公司名称，保障内部交易分类的准确性。

外币折算差异归零

内部交易红灯预警与巨额差异追责

月结关账期必须每日查看 ZP0JYADG0030 报表。一旦业务状态呈现红灯警告，月末系统将直接锁定，无法办理结账关账。各单位必须加强业财协作，及时冲销交易。
在年终决算对账中，若关联方内部交易对账差异达100万元以上，必须逐一查明，编写差异调节表及专项说明，并 提交公司总会计师签字确认。
