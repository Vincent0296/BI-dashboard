
## 一. 专项模块月结
### 一、 薪酬业务

- **薪酬业务月结**：
    
    - 完成本月薪酬业务计提、发放及相关凭证生成，包括工资、奖金、社保、公积金、福利费、工会经费及职工教育经费等业务处理 ；
        
    - 检查涉及薪酬业务的凭证中增减变动原因是否正确 ；
        
    - 检查用工类型是否选择正确，不同用工类型余额是否合理，是否存在串项情况 ；
        
    - 检查应付职工薪酬明细科目中是否存在不合理借方余额 。
        
- **薪酬报表检查**：在报表系统中取数并校验薪酬报表及相关损益表数据是否准确 。
    

### 二、 租赁业务

- **租赁业务月结**：完成租赁合同评估记账、当月折旧计提、资产清算对账、付款清算对账工作 。
    
- **租赁报表检查**：在报表系统中取数并校验租赁相关报表及相关资产负债表数据是否准确 。
    

### 三、 资产业务

- **资产业务月结**：完成资产业务单据检查、折旧检查、月末检查、资产提满折旧日期处理、数据一致性检查等工作 。
    
- **资产报表检查**：在报表系统中取数并校验资产相关报表及相关资产负债表数据是否准确 。
    

### 四、 往来资金业务

- **内部款项对账**：
    
    - 完成与上级资金单位内部往来科目对账工作 ；
        
    - 检查内部存款、内部存入款余额中贸易伙伴是否正确 。
        
- **往来类科目及报表检查**：
    
    - 通过客户、供应商及往来单位相关事务码检查往来单位挂账是否准确，是否有不合理余额（负数余额） ；
        
    - 检查往来类报表中账龄及逾期情况是否和业务实际情况相符 ；
        
    - 检查贸易伙伴是否正确 。
        
- **银行存款科目检查**：检查银行存款科目中现金流量项目、贸易伙伴及期限管理是否选择准确，是否有串项情况发生；检查不同账户是否存在贷方余额 。
    
- **内部交易**：每日通过“`ZP0JYADG0030`（内部交易实时情况监控）”事务代码检查内部往来交易和内部暂估交易，及早处理红灯预警以防月末无法关账 。
    

### 五、 总账业务

- **外币评估**：涉及外币业务的单位需在月末通过ERP系统标准化流程完成外币评估业务（汇率参考月末最后一天） 。
    
- **坏账计提**：本月应收款项数据无变化后，按照账面数据计提本月坏账金额，特别注意修改往来业务性质和是否逾期。
    
- **坏账报表检查**：在报表系统中取数并校验 BA07 减资准备情况表，检查 BA07 中本年计提数是否与利润表中信用减值损失和资产减值损失之和相等 。
    

### 六、 涉税业务

- **涉税业务月结**：完成税费业务计提及缴纳凭证的处理，检查增减变动原因是否正确，检查是否存在不合理借方余额 。
    
- **税费报表检查**：在报表系统中取数并校验税费相关报表，检查 BC02 与 BB41 的勾稽关系是否通过 。
    

### 七、 维修业务

- **维修业务停止**：完成本月所有维修业务，处理完毕需要本月结算的维修工单，并停止在系统中提报新的维修工单（未办完的顺延至下月） 。
    
- **维修工单月结**：财务部门完成维修工单的结算 。
    

### 八、 项目月结

- **投资项目结转**：检查本月投资项目涉及事项是否入账完成，完成后将投资项目结转至在建工程科目 。
    
 - **成本项目结转**：维修工单结转后，检查项目结转规则是否维护准确，核对项目发生额和实际业务是否相符，检查后将项目成本按照功能范围进行结转（功能范围6601和6602不用结转）。

## 二、 核心报表编制注意事项与勾稽关系

### 1. 资产负债表 (BA01)

- **合同资产错误**：严禁因合同负债清账错误导致往来单位出现借方余额，进而被系统自动重分类至合同资产 。
    
- **科目列报限制**：“一年内到期的非流动负债”除“一年内到期的租赁负债”外，其余其中项不允许有金额 。
    
### 2. 现金流量表 (BA04)

- **现金流项目**：按业务实质参考标准选择，严禁将代付款项错选至薪酬或分红支出 。
    
- **受限资金**：独立核算的党费户收付款，必须在期限管理中选择“C98-受限存款”，防止影响期末现金等价物余额 。
    
- **内部交易信息**：涉及中石油及华油内部交易时，必须在“银行存款”科目的往来单位和贸易伙伴字段维护对方信息，确保内外部分类准确 。
    
- **核心勾稽关系**：
    
    - 支付给职工的现金与应付职工薪酬表本年发放数勾稽 。
        
    - 支付的各项税费与应交税费表本年上交数勾稽 。
        
    - 个税手续费返还**不得**列入“收到的税费返还” 。
        

### 3. 减值准备情况表 (BA07)
- **增减变动原因**：常规计提/减少均需选择“A01001007-账龄分析法”，无特殊情况不得使用“其他原因增加/减少” 。
    
- **核心勾稽关系**：本年计提数必须与利润表中的“信用减值损失 + 资产减值损失”保持一致 。
    

### 4. 往来类报表

- **严禁负数余额**：往来报表中严禁出现负数余额 。逾期天数或账龄出现负数时，总账类科目可通过“FB02”更改到期日及付款条件调整；同时需及时进行清账操作 。
    

### 5. 税费及应付职工薪酬报表 (BB41/BB39)

- **税费结转原因**：月末结转增值税时，借方应交增值税-转出未交增值税选“S02001-本年减少-本年已交”；贷方未交增值税-本期转入数选“S01001-本年增加-本年应交” 。
    
- **用工类型**：准确选择以防报表中“合同化员工”或“市场化员工”出现负数 。
    
- **勾稽关系**：确保BC02（税金及附加）计提数与BB41实现数一致；BD03（人工计提数）与BB39（薪酬计提数）一致 。

## 三、 财务纪律与合规红线

- **严禁直接改表**：单体报表单位未经批准**严禁使用单体日记账直接调表** 。所有取数错误必须通过修改业务前端或ERP核算端数据解决，确保“账表相符” 。
    
- **外币折算差异归零**：自2026年起，华油集团全级次单位本位币均为人民币，本年报表中**外币报表折算差异不应有数** 。
    
- **内部交易红灯预警**：月结期间每日通过事务代码 `ZP0JYADG0030` 监控，**业务状态为红灯则月末无法关账**，须加强业财联动及时消除差异 。年终决算关联方对账差异在100万以上的，须逐一编写说明并提交总会计师签字 。
    
- **严控共享账务调整单**：实行严格审批流程（区域公司ERP总协调人审核 $\rightarrow$ 财务经理终审），确保操作留存记录、全程可追溯。财务部将按月通报使用情况 。
